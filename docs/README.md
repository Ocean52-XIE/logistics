## 方案概览

该培训网站面向物流公司内部员工，重点服务以下目标：

- 帮助新员工快速理解公司制度、物流流程、岗位职责与操作规范。
- 用课程、考试、闯关、案例和实操 SOP 的形式降低培训成本。
- 让主管、培训管理员、人事能够跟踪学习进度、考试结果和上岗准备度。
- 逐步沉淀企业自己的物流知识库、异常案例库和标准化培训体系。

## 推荐技术路线

- 前端：`Next.js + TypeScript + Tailwind CSS + Zustand + TanStack Query`
- 后端：`NestJS + TypeScript + PostgreSQL + Redis + MinIO/OSS`
- 管理与可观测性：`OpenAPI/Swagger + Winston/Pino + Prometheus + Grafana`
- 部署：`Docker + Nginx + CI/CD`

以上技术路线适合中小到中大型企业内部培训平台，优点是迭代快、前后端职责清晰、后续扩展移动端和小程序也比较容易。

## 建议优先级

第一阶段先上线以下闭环：

- 员工登录与角色权限
- 培训首页与学习路径
- 课程中心
- 在线考试
- 学习进度追踪
- 管理后台
- 培训报表

第二阶段再扩展：

- 岗位认证
- 实操签到与线下培训联动
- AI 问答助教
- 异常案例复盘中心
- 多组织、多仓、多城市培训体系
