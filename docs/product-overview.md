# 产品总览

## 1. 项目背景

物流公司新员工数量大、岗位类型多、业务变化快，传统线下培训通常存在以下问题：

- 培训资料分散，制度、SOP、岗位手册、异常案例难以统一管理。
- 新员工培训依赖师傅口传，质量不稳定，培训周期长。
- 培训完成情况难跟踪，管理层无法快速识别谁已经具备上岗能力。
- 考试与实际流程脱节，员工学完后仍可能不清楚收货、分拣、装车、签收、异常处理等关键步骤。

因此需要建设一个内部培训网站，统一承载课程、流程知识、考试考核、培训记录和运营分析，形成标准化培训闭环。

## 2. 项目目标

### 2.1 业务目标

- 将新员工平均上岗准备时间缩短 30% 以上。
- 让培训完成率、考试通过率、岗位认证完成率可视化。
- 降低培训管理员重复组织培训和人工统计的成本。
- 将公司流程、物流规则、安全规范和异常处理经验沉淀为长期资产。

### 2.2 系统目标

- 支持 Web 端统一学习与管理。
- 支持按岗位、部门、网点、仓库、城市配置培训内容。
- 支持图文、视频、PDF、流程图、考试题、案例题等多种内容形式。
- 支持从入职到上岗的完整学习路径。
- 支持培训效果追踪和管理报表导出。

## 3. 用户角色

### 3.1 员工

适用对象：

- 新入职员工
- 在岗复训员工
- 储备主管或班组长

核心诉求：

- 快速知道自己该学什么
- 能清楚查看培训进度和考试结果
- 能随时复习 SOP、流程和案例

### 3.2 培训管理员

核心诉求：

- 管理课程、试题、学习路径和培训计划
- 追踪员工学习完成情况
- 快速发布新制度和新流程培训

### 3.3 部门主管 / 站点负责人

核心诉求：

- 查看本部门员工培训状态
- 识别未达标人员
- 安排补训和复训

### 3.4 人事 / 运营管理层

核心诉求：

- 评估入职培训效率
- 监控不同组织培训质量
- 用数据支撑绩效、转正和岗位认证

## 4. 核心业务场景

### 4.1 新员工入职培训

典型流程：

1. 员工入职后自动分配“新员工通用学习路径”。
2. 系统根据岗位再追加对应课程，例如仓储操作员、分拣员、调度员、客服、司机。
3. 员工完成课程学习、章节测验和结业考试。
4. 主管查看学习结果，决定是否允许上岗或进入跟岗阶段。

### 4.2 流程更新培训

例如：

- 新增派送签收规范
- 仓库上架规则调整
- 异常件处理流程更新
- 安全生产制度更新

系统支持按部门或岗位定向推送必修培训，并要求限时完成。

### 4.3 岗位认证与复训

系统支持将培训与岗位认证绑定，例如：

- 收货岗认证
- 分拣岗认证
- 异常处理岗认证
- 客服工单处理认证

当制度变化、考试未通过或认证到期时，系统自动触发复训。

### 4.4 异常案例学习

围绕物流常见问题建立案例库，例如：

- 货损货差
- 错分错派
- 异常签收
- 延误预警
- 客诉升级

员工通过案例题学习“场景判断 + 标准处置流程”。

## 5. 功能范围

## 5.1 员工端功能

- 登录与身份识别
- 首页待办与学习路径
- 课程中心
- 知识库 / SOP 中心
- 在线考试
- 我的学习记录
- 证书 / 岗位认证
- 通知公告
- 常见问题与帮助

## 5.2 管理端功能

- 用户与组织管理
- 角色与权限管理
- 课程管理
- 题库管理
- 学习路径管理
- 培训计划管理
- 考试配置
- 数据报表
- 通知中心
- 内容审核与版本管理

## 5.3 后台运营功能

- 培训完成率统计
- 考试通过率统计
- 岗位维度培训分析
- 组织维度培训分析
- 高风险知识点分析
- 课程热度与失分点分析

## 6. 培训内容结构设计

建议将内容划分为 5 个一级模块：

1. 公司认知
   包括企业文化、组织架构、考勤制度、行为规范。
2. 物流基础
   包括物流网络、运输模式、仓配流程、签收与回单基础知识。
3. 岗位流程
   包括按岗位拆分的标准 SOP 和操作注意事项。
4. 安全与合规
   包括消防、安全生产、车辆安全、货物合规、信息安全。
5. 异常与案例
   包括异常件、投诉、延误、破损、拒收、赔付流程等。

每个模块下建议统一内容颗粒度：

- 课程
- 章节
- 知识点
- 小测
- 结业考试
- 实操检查项

## 7. 培训闭环

推荐的培训闭环如下：

1. 指派培训
2. 学习课程
3. 章节测试
4. 结业考试
5. 主管审核
6. 上岗认证
7. 数据复盘
8. 持续复训

这个闭环能让平台不仅“能看课”，还真正具备管理价值。

## 8. 非功能需求

- 支持高并发登录和统一培训发布。
- 页面在低配置办公电脑上也要流畅。
- 视频、PDF 等大文件加载稳定。
- 支持审计日志，满足内部管理要求。
- 支持数据备份与权限隔离。
- 支持中文全文检索，方便查 SOP。

## 9. 成功指标

建议重点跟踪以下指标：

- 新员工 7 日培训完成率
- 新员工首考通过率
- 上岗前培训达标率
- 课程平均完成时长
- 高频失分知识点
- SOP 搜索命中率
- 培训管理员单月组织效率提升

## 10. 推荐边界

本期建议先不纳入以下复杂能力，避免首版过重：

- 外部客户培训
- 多语言体系
- 复杂直播培训系统
- 完整绩效系统
- 薪酬联动

这些能力可以在平台稳定后逐步扩展。
